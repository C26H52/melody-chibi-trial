# Waving frame assets

This folder contains the exported PNG frames for the waving animation experiments.

- `04-waving-original-smile-restored-frames/`: frames from the restored waving GIF with the original smile.
- `04-waving-crooked-smile-frames/`: frames from the crooked-smile waving GIF variant.

The pixel-region transform helper is tracked at `../scripts/region_layer_transform.py`.
